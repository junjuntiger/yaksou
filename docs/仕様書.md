## 3. 画面別仕様

### S01 ログイン画面

| 項目 | 仕様 |
|------|------|
| 入力項目 | メールアドレス・パスワード |
| ボタン | ログイン・新規登録はこちら |
| バリデーション | 未入力時はFirebaseがエラー返却 |
| 成功時 | メインタブへ遷移 |
| 失敗時 | エラーメッセージをalertで表示 |

### S03 撮影画面

| 項目 | 仕様 |
|------|------|
| 操作 | 「カメラで撮影 / 写真を選択」ボタンでinput[type=file]を起動 |
| 対応形式 | image/* |
| AI識別 | `/api/identify` にPOSTリクエスト（Base64エンコード） |
| 識別中 | 「AI識別中...」メッセージを表示 |
| 識別成功 | 登録フォーム(S08)へ自動遷移（識別結果を初期値として設定） |
| 識別失敗 | エラーメッセージをalertで表示 |

### S04 みんなの投稿 / S05 マイ手帳（共通）

| 項目 | 仕様 |
|------|------|
| フィルタ | 全て・薬草・花・雑草のタブ切り替え |
| 検索 | 植物名・地域・撮影者名で部分一致検索（S04のみ撮影者名含む） |
| 表示 | 2カラムグリッド（PC）・1カラム（モバイル） |
| カード | 植物アイコン・植物名・カテゴリバッジ・効能・地域・日付・撮影者 |
| タップ | 詳細画面(S07)へ遷移 |
| データ | Firestoreの `plants` コレクション（リアルタイム同期） |

### S07 詳細画面

| 項目 | 仕様 |
|------|------|
| 表示項目 | 植物名・カテゴリ・撮影日・地域・撮影者・効能・使い方 |
| 修正ボタン | 自分の投稿のみ表示。クリックで編集フォーム(S09)へ遷移 |

### S08 登録フォーム / S09 編集フォーム

| 入力項目 | 型 | 必須 | 備考 |
|---------|---|------|------|
| 植物名 | テキスト | ○ | AI識別結果を初期値として設定 |
| カテゴリ | セレクト | ○ | 薬草・花・雑草 |
| 効能 | テキストエリア | - | AI識別結果を初期値として設定 |
| 使い方 | テキストエリア | - | AI識別結果を初期値として設定 |
| 地域 | テキスト | - | プロフィールの地域を初期値として設定 |
| 撮影日 | 日付 | ○ | 当日を初期値として設定 |
